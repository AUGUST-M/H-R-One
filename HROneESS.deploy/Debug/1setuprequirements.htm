Setup requirements >
Workstation
- It is recommended that you have a PC with these features:

1. Pentium IV 2.0G Hz microprocessor or above
2. Microsoft Windows 98,2000, XP or Vista
3. Microsoft Internet Explorer 6.0+ or Firefox 1.0+
4. VGA Monitor, preferably with 1024 x 768 resolution
5. Minimum 512MB RAM and 30 GB free hard disk space
6. Acrobat 6.0 with Asian font pack of Traditional Chinese for report viewing


Database & Web Server
- It is recommended that you have server with these features:

1. Pentium IV 3.0G Hz microprocessor or above
2. Microsoft Windows 2000 Server, XP Profession or 2003 Server or above
3. MS SQL Server 2000, MSDE or above
4. Web server IIS 5.0 or above
5. Minimum 3GB RAM and 300 GB free hard disk space
6. Network ready with open TCP/IP Port 1433 for connection to SQL Server
